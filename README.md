 CardioMap: Healthcare Data Analysis using Machine Learning
 Overview

CardioMap is a student-built educational project that explores structured healthcare datasets using basic machine learning concepts.

The project demonstrates a simple data analysis pipeline including data understanding, processing, and result interpretation.

 Datasets

- EMS dataset (emergency medical services data)
- Hospital dataset (health facility information)

Workflow

Data → Preprocessing → Analysis → Model Simulation → Results Interpretation

 Results

All outputs are stored in the `results/` folder.

 Purpose

This project is intended for learning how machine learning can be applied to structured healthcare data.
 Disclaimer

This project is for educational purposes only and is not a medical tool.
